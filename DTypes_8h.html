Detailed Description

Get the H5 data type from a C++ object.

Author
Jon Burr (jon.b.nosp@m.urr@.nosp@m.cern..nosp@m.ch)
Version
0.1
Date
2021-12-04
Copyright
Copyright (c) 2021

This header provides a traits struct called H5DType that provides a function for getting the H5::DataType from an object. To customise the behaviour for another type of object simply provide a specialisation of this struct.

A specialised H5DType struct must contain a function with one of the two following signatures:

  • static H5::DataType getType();
  • static H5::DataType getType(const T& val);

The first signature is for classes whose corresponding H5 DataType is known at compile time. The second is for classes that require information from an instance to determine the entire type.

The default implementation provided here will be valid for C++ primitives and any class which defines a member function with the signature static H5::DataType h5DataType(); if the type is known at compile time or H5::DataType h5DataType() const; if it is not.

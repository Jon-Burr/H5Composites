Define a H5::CompType for a given struct.

Parameters
DTYPEThe name of the H5::CompType variable to define
STRUCTThe struct type to adapt
MEMBERSThe members to include. This should be a complete list or the datatype will reserve too much space

The STRUCT type must be a trivial (i.e. POD) type otherwise the offsetof macro gives undefined behaviour. Given a struct

struct Particle {
int ID;
float pt;
float eta;
float phi;
};

The macro call

H5COMPOSITES_STRUCT_DTYPE(particleDType, Particle, ID, pt, eta, phi)
H5COMPOSITES_STRUCT_DTYPE
#define H5COMPOSITES_STRUCT_DTYPE(DTYPE, STRUCT, MEMBERS...)
Define a H5::CompType for a given struct.

would be equivalent to

static H5::CompType particleDType(sizeof(Particle))
particleDType.insertMember("ID", HOFFSET(Particle, ID), H5::NATIVE_INT);
particleDType.insertMember("pt", HOFFSET(Particle, pt), H5::NATIVE_FLOAT);
particleDType.insertMember("eta", HOFFSET(Particle, eta), H5::NATIVE_FLOAT);
particleDType.insertMember("phi", HOFFSET(Particle, phi), H5::NATIVE_FLOAT);
